# Security Policy

## Supported Versions

We currently provide security updates for the following versions of VentureNode:

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

Security is a top priority for VentureNode. If you discover a security vulnerability, we appreciate your help in disclosing it to us in a responsible manner.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them by emailing the core maintainers directly or utilizing GitHub's private vulnerability reporting feature on this repository.

We will acknowledge receipt of your vulnerability report as soon as possible and strive to send you regular updates about our progress. If the vulnerability is accepted, we will release a patch and publish a security advisory.
